Name

mk_unquote_list — Unquote list into positional parameters

Synopsis

mk_unquote_list quoted...

Description

Unquotes an internally quoted list, such as one produced by mk_quote_list, placing each item in the list into the positional parameters ($1, $2, etc.).

Examples

# This is a no-op
mk_quote_list "$@"
mk_unquote_list "$result"
