Name

mk_chain_generic_stage — Generic stage function for chaining

Synopsis

MODULES="... chain-generic ..."

mk_chain_generic_stage target outdirs...

Description

This is the default stage function used by mk_chain_generic. It generates target by looking for it in each directory in outdirs in turn and copying from the first one to contain it. If it is not found in any, it aborts.
