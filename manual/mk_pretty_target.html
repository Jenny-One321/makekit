Name

mk_pretty_target — Convert fully-qualified target to pretty form

Synopsis

MODULES="... core ..."

mk_pretty_target target

Description

Converts target to a more compact form if possible, acting essentially as the inverse of mk_resolve_target. It will also normalize the path to remove intermediate . and .. components. Sets result to the result.
