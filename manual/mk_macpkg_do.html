Name

mk_macpkg_do — Begin Mac package definition

Synopsis

mk_macpkg_do PACKAGE=name VERSION=ver PMDOC=doc

Options

PACKAGE=name

Sets the name of the package

VERSION=ver

Sets the version of the package

PMDOC=doc

Specifies the PackageMaker document that describes the distribution. The XML files contained within will be processed as if by mk_output_file.

RESOURCES=res_list

Specifies a list of resource files relative to the directory containing doc that might be used. Each resource will be processed as if by mk_output_file.

BINRESOURCES=bres_list

Specifies a list of binary resource files relative to the directory containing doc. Unlike res_list, these files will used verbatim without processing.

Description

Begins the definition of a Mac distribution file. A distribution wraps one or more inner packages. You must specify one or more of these inner packages with mk_macpkg_sub_do and mk_macpkg_sub_done.

Use mk_macpkg_done to complete the definition.
