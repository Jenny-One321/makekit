Name

mk_are_same_path — Test if two paths are identical

Synopsis

mk_are_same_path path1 path2

Description

Returns 0 (logical true) if path1 and path2 represent the same physical file system path. Returns 1 (logical false) if they do not, or if either path is invalid.
