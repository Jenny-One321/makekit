Name

mk_parse_params — Parse special parameters

Synopsis

mk_parse_params

Description

Processes any positional parameters that fit the following forms:

VAR=value

Sets VAR to value.

@VAR={ val1 val2 ... }

Sets VAR to a quoted, space-separated list of values.

--

Treat subsequent parameters as ordinary.

Remaining parameters will be shifted such that $1 becomes the first ordinary parameter.

This should be used in combination with mk_push_vars and mk_pop_vars to ensure variables you expect to be passed are set only for the duration of the function.
