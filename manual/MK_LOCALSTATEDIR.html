Name

MK_LOCALSTATEDIR — Local state directory

Synopsis

MODULES="... path ..."
mk_declare -e MK_LOCALSTATEDIR

Description

The directory where programs should store machine-local state information.

Defaults to $MK_PREFIX/var, unless MK_PREFIX is /usr, in which case it defaults to /var.
