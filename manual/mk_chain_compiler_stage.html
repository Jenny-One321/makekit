Name

mk_chain_compiler_stage — C/C++-oriented stage function for chaining

Synopsis

MODULES="... chain-compiler ..."

mk_chain_compiler_stage target outdirs...

Description

This is the default stage function used by mk_chain_compiler. It understands how to generate missing .la files and merge multiple outputs into one binary on fat binary systems.
