Name

mk_moonunit — Build a MoonUnit test library

Synopsis

MODULES="... moonunit ..."

mk_moonunit DLO=dlo SOURCES=sources options...

Options

DLO=dlo

The name of the library

SOURCES=sources

The test sources

...

All options applicable to mk_dlo.

Description

Defines a target to build a MoonUnit test library. This function behaves nearly identically to mk_dlo, except that moonunit-stub is automatically invoked on the source files to generate a stub for the MoonUnit test loader, and INSTALLDIR defaults to @$MK_MOONUNIT_DIR.

To use this function, you must use first perform a configuration check with mk_check_moonunit. If you do not, or MoonUnit is not found, this function will fail.

Sets result to the generated library file target.
