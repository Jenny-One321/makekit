Name

MK_LIBEXECDIR — Install directory for program executables

Synopsis

MODULES="... path ..."
mk_declare -e MK_LIBEXECDIR

Description

The directory where programs intended for internal use by other programs or libraries are installed.

Defaults to $MK_EPREFIX/libexec.
