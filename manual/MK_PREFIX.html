Name

MK_PREFIX — Install prefix for architecture-independent files

Synopsis

MODULES="... path ..."
mk_declare -e MK_PREFIX

Description

The filesystem prefix for architecture-indepenent files. Used to construct the default install directories for headers, shared data files, configuration files, etc.
