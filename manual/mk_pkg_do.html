Name

mk_pkg_do — Begin Solaris package definition

Synopsis

mk_pkg_do PACKAGE=name VERSION=ver

Options

PACKAGE=name

Sets the name of the package

VERSION=ver

Sets the version of the package

Description

Begins the definition of a Solaris .pkg file. A .pkg file is a tar-like archive which contains one or more actual Solaris packages. You must specify one or more of these "subpackages" with mk_pkg_sub_do and mk_pkg_sub_done.

Use mk_pkg_done to complete the definition.
