Name

mk_lpp_do — Begin AIX package definition

Synopsis

mk_lpp_do PACKAGE=name VERSION=ver

Options

PACKAGE=name

Sets the name of the package. AIX package names are a hierarchy of dot-separated components, usually of the form vendor.product. If the name contains hyphens, as is typical of other package managers, they will be converted to dots.

VERSION=ver

Sets the version of the package. AIX package versions consist of 4 dot-separated numeric components.

Description

Begins the definition of an AIX LPP package file (.bff). You must define one or more filesets with mk_lpp_sub_do.

Use mk_lpp_done to complete the definition.
