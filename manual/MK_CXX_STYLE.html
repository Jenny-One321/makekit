Name

MK_CXX_STYLE — Style of C++ compiler

Synopsis

MODULES="... compiler ..."
mk_declare -e -s MK_CXX_STYLE

Values

none

No working C++ compiler is available

gcc

The C++ compiler is g++ or g++-compatible

unknown

The C++ compiler style is unknown

Description

Like MK_CC_STYLE, but for the C++ compiler.
