Name

MK_MULTIARCH — Multiarch style for target system

Synopsis

MODULES="... platform ..."
mk_declare -e -s MK_MULTIARCH

Values

none

not supported

separate

separate binaries for each ISA

combine

"fat" binaries combining versions for all ISAs

Description

The style of multiarch support for the target system.
