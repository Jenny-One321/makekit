Name

chain-compiler — Chain into foreign C/C++ build systems

Synopsis

MODULES="... chain-compiler ..."

Description

The chain-compiler module extends the chain-generic module to integrate non-MakeKit C/C++ components into your project. See chain-autotools for integrating autotools-based components in particular.
