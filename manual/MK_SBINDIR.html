Name

MK_SBINDIR — Install directory for system executables

Synopsis

MODULES="... path ..."
mk_declare -e MK_SBINDIR

Description

The directory where programs intended for use by the operating system or root user are installed.

Defaults to $MK_EPREFIX/sbin.
