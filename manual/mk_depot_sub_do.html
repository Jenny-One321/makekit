Name

mk_depot_sub_do — Begin HP-UX fileset definition

Synopsis

mk_depot_sub_do SUBPACKAGE=name

Options

SUBPACKAGE=name

Sets the name of the fileset

Description

Begins the definition of a fileset. This will create an output variable of the form DEPOT_FILESET_NAME which can be substituted into the correct place in the template PSF file specified to mk_depot_do.

After invoking this function, you can use functions such as mk_package_targets or mk_package_patterns to add files to the fileset. End the definition with mk_depot_sub_done.
