Name

MK_CXX — C++ compiler

Synopsis

MODULES="... compiler ..."
mk_declare -e -s MK_CXX

Description

Set to the executable and arguments used to invoke the C++ compiler. Note that this variable is system-dependent and may have separate values for each target system and ISA.
