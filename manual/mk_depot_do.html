Name

mk_depot_do — Begin HP-UX package definition

Synopsis

mk_depot_do PACKAGE=name PSF=psf

Options

PACKAGE=name

Sets the name of the package

VERSION=ver

Sets the version of the package

PSF=psf

Designates a template product specification file to use.

CONTROLFILES=cfiles

Specifies a set of control files which may be referenced by the product specification These will be processed as by mk_output_file and placed in a directory which may be referenced from the product specification as @DEPOT_CONTROL@.

Description

Begins the definition of an HP-UX package to be built. You must provide a template product specification file which lays out the structure and metadata of your distribution. The file will be processed as by mk_output_file. You must specify one or more filesets with with mk_depot_sub_do.
