Name

mk_check_sizeofs — Check sizes of several types

Synopsis

MODULES="... compiler ..."

mk_check_sizeofs types...

Description

Runs mk_check_sizeof on each type in types. For each type, the variable SIZEOF_TYPE will be set to the result, and SIZEOF_TYPE will be similarly defined in the current config header.
