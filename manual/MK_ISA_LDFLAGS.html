Name

MK_ISA_LDFLAGS — Per-ISA linker flags

Synopsis

MODULES="... compiler ..."
mk_declare -e -i -s MK_ISA_LDFLAGS

Description

The flags in this variable will be passed to any compiler operation involving linking an executable, library, or dynamically-loadable object. Because it is a system-dependent variable, it may be used to set flags that apply to only one particular target system and ISA.
