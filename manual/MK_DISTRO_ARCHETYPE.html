Prev Core Functionality Next

Name

MK_DISTRO_ARCHETYPE — Target OS distribution archetype

Synopsis

MODULES="... platform ..."
mk_declare -e -s MK_DISTRO_ARCHETYPE

Values

redhat

Red Hat-like (RHEL, CentOS, Fedora...)

debian

Debian-like (Debian, Ubuntu...)

suse

SuSE-like (SLED, SLES, OpenSuSE...)

Description

The OS distribution archetype of the current target system. This is more abstract than the precise distribution.
