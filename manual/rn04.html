Chaining Modules

The chain-autotools module extends the chain-compiler module to integrate autotools (autoconf, automake, libtool) components into your project.

The chain-compiler module extends the chain-generic module to integrate non-MakeKit C/C++ components into your project. See chain-autotools for integrating autotools-based components in particular.

The chain-generic module allows you to chain into the build system of a foreign component embedded in your project. This is a generic module designed for any component; see chain-compiler for C/C++ components and chain-autotools for autotools components.
