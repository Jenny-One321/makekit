Name

mk_group — Build a source group

Synopsis

MODULES="... compiler ..."

mk_group GROUP=name options...

Description

Defines a target to build a C/C++ "group", which combines source files and associated build flags into a logical unit which can be added to mk_program and friends. This feature is similar to "convenience libraries" with GNU libtool.

See the compiler module for common options or Groups in the MakeKit guide for usage examples.
