Core Functionality

The core module provides basic functionality for defining build rules and running configure tests.

This module detects and manages configuration of installation paths for programs, libraries, documentation, and other types of files.

This module detects properties of the build and host system, including CPU architecture and its supported instruction set architectures, operating system, and distribution. It allows switching the target system and ISA for subsequent tests or build rules.

This module provides functions for locating programs on the build system.
