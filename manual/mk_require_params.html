Name

mk_require_params — Check for missing keyword parameters

Synopsis

mk_require_params funcname params..

Description

Checks that each variable in params is not the empty string or fails with an error message. The provided funcname will be used in the message to identify to the user which function expected the missing parameter.

This function is intended for use after mk_parse_params in order to verify that all non-optional parameters were passed.
