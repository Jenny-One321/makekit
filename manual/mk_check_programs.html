Name

mk_check_programs — Check for programs on build system

Synopsis

MODULES="... program ..."

mk_check_programs programs...

Options

FAIL=yes|no

If set to yes, fails configuration if any program is not found. Defaults to no.

Description

Checks for a list of available programs by passing each to mk_check_program.
