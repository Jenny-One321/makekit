Name

chain-generic — Chain into foreign build systems

Synopsis

MODULES="... chain-generic ..."

Description

The chain-generic module allows you to chain into the build system of a foreign component embedded in your project. This is a generic module designed for any component; see chain-compiler for C/C++ components and chain-autotools for autotools components.
