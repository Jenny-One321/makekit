Name

mk_check_doxygen — Check for Doxygen on the build system

Synopsis

mk_check_doxygen

Description

Checks for the availability of Doxygen on the build system. The result can be tested with mk_have_doxygen. If successful, you may then use functions such as mk_doxygen_html to build Doxygen documentation as part of your project.
