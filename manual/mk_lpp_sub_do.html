Name

mk_lpp_sub_do — Begin AIX fileset

Synopsis

mk_lpp_sub_do SUBPACKAGE=name DESCRIPTION=desc

Options

SUBPACKAGE=name

Sets the name of the fileset. This will be appended to the name of the overall package.

VERSION=ver

Sets the version of the fileset. Defaults to the version specified for the overall package.

DESCRIPTION=desc

Sets a brief English-language description.

PREIN=prein

Indicates a script file that should be run before installing the fileset. All scripts are processed as by mk_output_file.

UNPREIN=unprein

Indicates a script file that should be run to revert the effects of the prein script.

POSTIN=postin

Indicates a script file that should be run after installing the fileset.

UNPOSTIN=unpostin

Indicates a script file that should be run to revert the effects of the postin script.

PRERM=prerm

Indicates a script that should be run before removing a previous version of the fileset.

CONFIG=config

Indicates a script that should be run when configuring the fileset.

UNCONFIG=unconfig

Indicates a script that should be run when unconfiguring the fileset.

REQUISITES=reqfile

Specifies an optional requisites file that describes the dependencies of the fileset.

UPSIZE=upspec

Specifies a list of directories and how much space in 512-byte blocks might need to be allocated in order to install the fileset, e.g. "/opt 2048".

Description

Begins the definition of a fileset within an AIX package. Use functions such as mk_package_patterns to include files in the package.

By default, intermediate directories for included files are assumed to exist on the target system to avoid potentially overriding their permissions. If you install files into directories that are not pre-existing on the system, you should specify them with mk_package_dirs.

Use mk_lpp_sub_done to complete the definition.
