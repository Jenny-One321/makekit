Name

mk_defname — Convert a string to #define form

Synopsis

MODULES="... compiler ..."

mk_defname str

Description

Converts a string to a form suitable for use as a variable name or #define. This implements the same rules that autoconf uses: all letters are uppercased, all non-letter, non-number characters are converted to _, except for *, which is converted to P. Sets result to the result.
