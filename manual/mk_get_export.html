Name

mk_get_export — Get value of exported variable

Synopsis

MODULES="... core ..."

mk_get_export dir var

Description

Gets the value of the exported variable var in the directory dir (relative to the directory containing the current MakeKitBuild). The specified directory must have been previously processed.

The value is placed in result.
