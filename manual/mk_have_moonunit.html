Name

mk_have_moonunit — Test if MoonUnit was found

Synopsis

MODULES="... moonunit ..."

mk_have_moonunit

Description

Returns 0 (logical true) if MoonUnit was found successfully by mk_check_moonunit, and 1 (logical false) otherwise.
