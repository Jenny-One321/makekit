Name

MK_CC_LD_STYLE — Linker style for C

Synopsis

MODULES="... compiler ..."
mk_declare -e -s MK_CC_LD_STYLE

Values

gnu

GNU ld or compatible

native

Native OS linker

Description

Defines the style of the linker used for linking objects derived from C code. This is an abstract classification which encompasses the behavior and supported parameters of the linker.
